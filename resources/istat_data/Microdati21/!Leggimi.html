Il file ISTAT_MFR_ECH_2021_IT_DELIMITED.zip contiene, in due directory dedicate, i file di seguito elencati:
file di microdati, tracciato record con classificazioni associate, documenti metodologici.
Il nome di ciascun file è caratterizzato dalla tipologia del file (MFR), dall'acronimo dell'indagine, dal contenuto dello stesso, dal periodo di riferimento dei dati e dall'eventuale area tematica. Per quanto riguarda i file di microdati ed il relativo tracciato, nel nome viene evidenziato anche il formato: con campi a lunghezza fissa (Fixed) oppure con campi separati dal catarattere '^' (Delimited).
Per quanto riguarda le classificazioni, esse sono raggruppate nella cartella METADATI/Classificazioni. Il nome del file di ciascuna classificazione è contrassegnato dal suffisso var seguito dal numero d'ordine presente nel tracciato record relativo alle variabili di tipo categorico.
